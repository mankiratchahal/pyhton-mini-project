# 🐍 Python Mini Projects Collection

A collection of beginner-friendly Python projects demonstrating core programming concepts such as:

- Conditional Statements (`if`, `elif`, `else`)
- Looping Statements (`for`, `while`)
- Lists
- Tuples
- Dictionaries
- Sets
- User Input Handling
- Menu-Driven Programming

---

## 📂 Projects Included

### 1. 🎯 Number Guessing Game

A simple game where the computer generates a random number and the player tries to guess it.

#### Features
- Random number generation
- Unlimited guessing attempts
- Hints (Too High / Too Low)
- Attempt counter

#### Concepts Used
- `while` loop
- `if-else` statements
- `random` module
- User input

---

### 2. 🎓 Student Result Management System

A menu-driven application for managing student academic records.

#### Features
- Add student records
- Calculate total marks and percentage
- Generate grades automatically
- Search students by roll number
- Display all student results
- Show unique grades

#### Concepts Used
- Lists (marks)
- Tuples (result summary)
- Dictionaries (student records)
- Sets (unique grades)
- Loops and conditional statements

---

### 3. 📚 Library Management System

A simple library management application to manage books and issue/return operations.

#### Features
- Add books
- View available books
- Search books
- Issue books
- Return books
- Display unique authors

#### Concepts Used
- Lists
- Tuples
- Dictionaries
- Sets
- Loops and conditions

---

## 🛠️ Technologies Used

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-mini-projects.git
```

2. Navigate to the project folder:

```bash
cd python-mini-projects
```

3. Run any project:

```bash
python number_guessing_game.py
```

or

```bash
python student_result_management.py
```

or

```bash
python library_management_system.py
```
